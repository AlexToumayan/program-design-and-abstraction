Installing Python on macOS

◷ approx. 4 min read

This guide takes you from a brand-new Mac to a working python3 command, one step at a time. You do not need any prior setup, and you cannot break anything by following along. If a step does not go as described, jump to When something goes wrong at the bottom — it covers the handful of snags that trip up almost everyone.

Step 1 — Open the Terminal

Everything below happens in the Terminal, a plain text window where you type one instruction, press Return, and the Mac answers. To open it:

  • Press Command-Space to open Spotlight, type Terminal, and press Return.
  • (Or open Finder → Applications → Utilities → Terminal.)

A window appears with a line ending in a % or $ symbol and a blinking cursor. That symbol is the prompt — the Mac saying "I'm ready; type a command." You never type the prompt itself; you type what comes after it. The examples below show this prompt as a $; if yours is a %, treat the two as the same — either way, type only what follows it.

Step 2 — Check whether Python is already installed

Macs do not ship with the Python you want pre-installed, but it is worth a five-second check. At the prompt, type this and press Return:

$ python3 --version
  • If you see a line like Python 3.12.2 — any number that begins with 3 — Python is already installed. Skip ahead to Step 4.
  • If you see command not found, or a pop-up offers to install the "command line developer tools," you do not have it yet. If that pop-up appeared, click Cancel to close it for now, then continue to Step 3.

Step 3 — Download and install Python

The simplest path for a first install is the official installer from python.org:

  1. Download. In your web browser, go to python.org/downloads. The page detects that you are on a Mac and shows a large button to download the latest Python 3. Click it and wait for the .pkg file to finish downloading.
  2. Run the installer. Open the downloaded .pkg file. A window walks you through a few screens — click Continue, agree to the license, and click Install. macOS may ask for your password; that is normal. There is nothing tricky to tick on a Mac.
  3. Finish. When it reports success, close the installer and move the downloaded file to the Trash.

(If you already use Homebrew, brew install python3 works just as well. If that sentence meant nothing to you, ignore it — the installer above is all you need.)

Step 4 — Confirm it works

Close your Terminal window and open a fresh one (Step 1) so it picks up the new install. Then ask for the version again:

$ python3 --version
Python 3.12.2

Any answer starting with 3 means you are ready. Now start Python itself by typing its name with nothing after it:

$ python3
>>>

The prompt changes to >>>, which means Python is now listening for code. Try one line:

>>> 2 + 2
4

If Python answers 4, your setup is complete. To leave Python and return to the ordinary $ prompt, type quit() and press Return (or press Control-D).

When something goes wrong

  • command not found: python3 even after installing. You are almost always looking at an old Terminal window opened before the install finished. Close every Terminal window, open a fresh one, and try again.
  • python works but python3 does not, or the other way around. On a Mac the command you want is almost always python3; plain python may be missing or point at a very old version. Throughout the course, use python3.
  • A pop-up about "command line developer tools." Step 2 has you click Cancel on this and install from python.org instead, which gives you the newest Python. Clicking Install on the pop-up also works if you prefer the quicker route, though it can be a slightly older version.
  • The version that appears starts with 2. That is the system's leftover Python 2, not what we use. Install from python.org (Step 3) and keep typing python3.

Once python3 --version answers with a 3, you are done here — head back to 1.1 Getting Started and pick up where you left off.
